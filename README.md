# UNP_SQL_PROJECT_E_COMMERCE_STORE_DATASET_ANALYSIS

DESCRIPTION: The E-commerce Database Project is designed to manage and organize online shopping data efficiently using MySQL. This project includes tables for customers, products, orders, payments, and order details. It helps store important information such as customer details, product prices, stock availability, and order history. By using SQL queries, the system can insert, update, delete, and retrieve data easily. The database also maintains relationships between tables using primary keys and foreign keys to ensure data accuracy and integrity. This project demonstrates how a real-world online shopping platform stores and manages its data in a structured way..
